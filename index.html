UpFuckr by Matt Katz

An Android uploader for FuckFlickr

FuckFlickr is an open-source image gallery that won't narc you out. No database, just files and YAML. Easy.

UpFuckr is an Android application that lets you upload from your Gallery to your own sweet FuckFlickr install.

See the ditz html pages for a roadmap - ditz is used for bugs and planning. Done in vim only as eclipse doesn't run fast on my tiny gerbil powered laptop.

If you like this, you will probably also want to try Flickr Lube - it makes it easier to FuckFlickr.

Dependencies

You need to have AndFTP from Lysesoft installed - it does all the ftp work for us.
